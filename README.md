# PageTalker 📄💬
 PageTalker is an AI SaaS platform where users can chat with their PDF, building using NextJs 13, React, Tailwind , Prisma and tRPC

## 🌟📃 Full Feature List:
🛠️ Complete SaaS Built From Scratch<br/>
💻 Beautiful Landing Page & Pricing Page Included<br/>
🔒 Authentication Using Clerk<br/>
💳 Free & Pro Plan Using Stripe<br/>
⚡ Infinite Message Loading for Performance<br/>
📄 A Beautiful And Highly Functional PDF Viewer<br/>
🔄 Streaming API Responses in Real-Time<br/>
🎨 Clean, Modern UI Using 'shadcn-ui'<br/>
🚀 Optimistic UI Updates for a Great UX<br/>
📤 Intuitive Drag n’ Drop Uploads<br/>
✨ Instant Loading States<br/>
🔧 Modern Data Fetching Using tRPC & Zod<br/>
🧠 LangChain for Infinite AI Memory<br/>
🌲 Pinecone as our Vector Storage<br/>
📊 Prisma as our ORM<br/>
🔤 100% written in TypeScript<br/>
